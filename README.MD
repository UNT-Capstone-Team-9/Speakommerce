# Speakommerce

## Team
CLOUD9

## Course
CSCE 4901 Capstone I

## Project Overview
Speakommerce is an accessibility focused ecommerce navigation system designed to support visually impaired users through voice based browsing and search. The project spans two semesters with analysis and design completed in Capstone I and implementation and demonstration completed in Capstone II.
